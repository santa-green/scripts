Підписання документів:
1. Виконуємо ініціалізацію ком-об’єкта – метод Initialize:
caType = "UA1";    
Initialize(caType);
2. Встановлюємо графічий режим бібліотеки = False:
SetUIMode(False);
3. Виконуємо знищення в пам’яті особистого ключа (для бухгалтера, директора, печатки) – метод ResetPrivateKey, в який передається відповідний тип ключа.
4. встановлюємо признак необхідності використання транспортного та криптозаголовків = False:
useHeaders = False;
SetFilesOptions(useHeaders);
5. Встановлюємо ключ бухгалтера або директора – метод SetPrivateKey, в який передається відповідний тип ключа
6. Встановлюємо ключ печатки – метод SetPrivateKey.
7. Наносимо на файл (або масив файлів) підпис бухгалтера – метод SignFilesByAccountant.
Або наносимо підпис директора – метод SignFilesByDirector.
8. Наносимо на файл (або масив файлів) підпис печатки – метод SignFilesByDigitalStamp.
